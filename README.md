## ✨ Features

| Feature | Description |
|---|---|
| 📖 All 114 Surahs | Arabic text with audio recitation |
| 🧠 Hifdh Tracker | Memorisation progress and focus mode |
| 📝 Tajweed Quizzes | 80+ questions across multiple levels |
| 🌙 Daily Āyah | Daily verse with reflection |
| 🕌 Salah Times | Prayer times by city (GPS or manual) |
| 🎓 Certificate Generator | Printable certificates for students |
| 📋 Homework Sheets | Downloadable homework for students |
| 👨‍🏫 Teacher Mode | Class management and student tracking |
| 🏆 Progress Tracking | Points, streaks, and achievements |
| 📵 Offline First | Works without internet after first load |

## 🚀 How to Use

**Option 1 — Use Online**  
Visit the live link and use it directly in your browser — nothing to download.

**Option 2 — Download and Use Offline**  
1. Download `index.html` from this repository
2. Open it in any browser (Chrome, Firefox, Safari)
3. That's it — no setup, no login, no internet needed

---

## 👨‍🏫 Teacher Mode

Teacher Mode gives full access to:
- Class and student management
- Assigning homework to students
- Generating Certificates of Achievement
- Tracking student progress and stages

Access it by tapping the teacher icon on the main screen and entering your passcode.
